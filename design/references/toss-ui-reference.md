# 토스 UI 디자인 레퍼런스

## 핵심 디자인 원칙

- **명확함**: 사용자가 다음에 무엇을 해야 하는지 항상 명확하게
- **간결함**: 한 화면에 하나의 목적만
- **신뢰감**: 금융 서비스답게 안정적이고 깔끔한 느낌

## 컬러

- Primary: `#3182F6` (토스 블루)
- Background: `#F9FAFB` (밝은 회색)
- Text Primary: `#191F28`
- Text Secondary: `#8B95A1`
- Success: `#3ED4A4`
- Error: `#F04452`

## 타이포그래피

- 폰트: Pretendard (한국어 UI 기본)
- 제목: 20~24px, Bold
- 본문: 15~16px, Regular
- 보조텍스트: 13~14px, Regular

## UI 패턴

- 버튼: 둥근 모서리 (border-radius: 12px), 하단 고정 CTA
- 카드: 흰 배경 + 약한 그림자, 모서리 16px
- 입력폼: 하단 라인만 있는 심플한 스타일
- 간격: 4px 단위 시스템 (8, 16, 24, 32px)
